Data Journalism Tools and Resources

Note/disclaimer: these are not the only tools/resources available and you do not have to use them all. See what fits your work and keep looking for updates or for new ones. Resources are not in full order. Most standard commercial software not listed.
You can find more resources using the hashtag #ddj and #DataJournalism (in Arabic: #صحافة_البيانات). Twitter usually takes a few minutes to retrieve recent tweets, so check for new tweets later.

Tools: Data Acquisition and Cleaning
  • Tabula: extract tables from PDF files
  • OpenRefine: work with messy data (see also Data Wrangler (commercial: Trifacta). See also: Open Refine tutorial , cleaning Arabic data with OpenRefine, this video and this School of data article
  • Import.io: extract web data online
  • Scraping for journalism: ProPublica data collection guide
  • Scrapy: extract data from websites (uses Python)
  • Portia: Scrape websites visually. No code required.
  • OutWit hub: collect and organize data and media (lite version is free)
  • Web Scraper: Chrome extension to scrape data from the web and save as csv file
  • Scraper Wiki: tools and services for data extraction and more (see PDF tables for getting data from PDF)
  • Apache Tika: extract text and metadaa from various file types
  • MS Excel and Google Sheets: both can import basic tables and have ways to deal with messy data (ex. Import in Google docs)
  • Scraping without programming: NICAR 2016 presentation by Samantha Sunne
  • Application APIs: most popular data application (ex. Twitter, Facebook, YouTube, ... ) have their own APIs for data extraction.
  • Programming and scripting: ex. using Python or R and writing code
  • OCHA OPT: database for Gaza crossings
  • Foreign assistance data: explore map or search data
  • Google Search: tips for journalists
  • Datasets: general collections (see also map of data portals and IATI Aid to Palestine data)
  • Knoema: datasets by topic and country (link is for Palestine)
  • Top 30 free data scrapers:
  • Data is Plural: has archives of newsletters and datasets
Data Analysis
  • Google sheets: free online, excel-like, requires gmail account
  • MS Excel: popular, part of MS Office (see also: Power BI and power viewer)
  • Data Journalism - visualization (by Alberto cairo)
  • Tableau Public: free version (limited) of commercial application - analysis and visualization
  • Workbench: analyze and visualize data without coding
  • Programming languages like Python and R (require coding) and other commercial software and services
Data Visualization
  • Google fusion tables: web app to gather and visualize data
  • Google public data (data and visualization)
  • D3.js: Java script library for data driven documents (example tutorial)
  • Power BI: free online service from Microsoft, links to many data sources
  • Data Wrapper: create charts online
  • Carto: location data visualization and analysis (previously called cartoDB)
  • Infogram: create charts and infographics online. Similar tools include easel.ly, Pictochart, visual.ly, Visme and Canva.
  • Knight Lab projects (storytelling)
  • Cytoscape: networkd analysis and visualization
  • Graph Commons: network mapping
  • Timeline.js: easily made timelines
  • Silk: online interactive visualization
  • Piktochart: easy to use infographics maker
  • Tableau Public: free version (limited) of commercial application - analysis and visualization (note: Tableau acquired InfoActive in Aug. 2015
  • Plotly: charts, visualization and dashboards
  • Canva: general purpose design tool
  • Keshif: from Univ. of Maryland
  • Programming languages like Python and R (require coding) and other commercial software and services
