Data Journalism Tools and Resources

Note/disclaimer: these are not the only tools/resources available and you do not have to use them all. See what fits your work and keep looking for updates or for new ones. Let us know if you have a new important resource or if you notice a bad or broken link here. Resources are not in full order. Most standard commercial software not listed.
You can find more resources using the hashtag #ddj and #DataJournalism (in Arabic: #صحافة_البيانات). Twitter usually takes a few minutes to retrieve recent tweets, so check for new tweets later.

Tools: Data Acquisition and Cleaning
  • Tabula: extract tables from PDF files. If you need to extract pages from a PDF file and you do not have Adobe pro (writer), you can use a page extaction tool (must support Arabic). Other Tools for data extraction from PDF files are also available. See also Tabulizer in R.
  • OpenRefine: (used to be Google Refine) - work with messy data (see also Data Wrangler (commercial: Trifacta). See also: Open Refine tutorial, solving data impoprt problems with Open Refine (ex. from tabula), , cleaning Arabic data with OpenRefine, this video and this School of data and Clean Your Data (getting started with OpenRefine - make sure you check the Resouorces page for a good list of helpful links)
  • Import.io: extract web data online (professional but expensive, 7 day trial, edu/journalism discount)
  • Scraping for journalism: ProPublica data collection guide
  • Scrapy: extract data from websites (uses Python)
  • ** Portia: Scrape websites visually. No code required. See this tutorial on Portia and other tools. A similar tool is Instant data scraper (very simple, Chrome extension)
  • Top 20 data extraction tools. Some tools like Octoparse and ParseHub offer free/lite versions, Webharvy is a Windows application (paid) - (ParseHub Tutorials, Octoparse tutorials, Webharvy).
  • OutWit hub: collect and organize data and media (lite version is free)
  • Web Scraper: Chrome extension to scrape data from the web and save as csv file
  • Scraper Wiki: tools and services for data extraction and more (see PDF tables for getting data from PDF)
  • Apache Tika: extract text and metadaa from various file types
  • MS Excel and Google Sheets: both can import basic tables and have ways to deal with messy data (ex. Import in Google docs)
  • Scraping without programming: NICAR 2016 presentation by Samantha Sunne
  • Application APIs: most popular data application (ex. Twitter, Facebook, YouTube, ... ) have their own APIs for data extraction.
  • Programming and scripting: ex. using Python or R and writing code. Example: Scraping in Python Part 1 and Part 2
  • OCHA OPT: database for Gaza crossings
  • Foreign assistance data: explore map or search data
  • Google Search: tips for journalists
  • Datasets: general collections (see also map of data portals and IATI Aid to Palestine data)
  • Knoema: datasets by topic and country (link is for Palestine)
  • Top 30 free data scrapers:
  • Data is Plural: has archives of newsletters and datasets
  • Open Semantic Search (search and language tools, accepts fuzzy searches, Arabic included)
  • Kaggle popular datasets - see examples of possible topics and data
  • Quartz Directory - essential data
  • Foreign Lobby Watch - search foreign entities in USA
  • Global Stats - by theme, EU and Global
  • Arab Development Portal - visualize or export data in different categories
  • ESRI Demographics - many countries, Palestine not included.
  • How Much (data and visualization in different topics). See also Our World in Data.
  • data.world - collaboration and networking around data analysis

Data Visualization
  • Google fusion tables: web app to gather and visualize data
  • Flourish - from Google, create visuals without coding
  • Animated gifs tutorial (video like)
  • Google public data (data and visualization)
  • D3.js: Java script library for data driven documents (example tutorial)
  • Power BI: free online service (sample live visual) from Microsoft with desktop version, links to many data sources. You can also create infographs using a free custom visual for Power BI (need school or work email to download) - see this video tutorial. Also, try this blog about Power BI for custom visuals and more. ( M language reference, Blogs, M resources)
  • Data Wrapper: create charts online
  • Carto: location data visualization (mapping) and analysis online (previously called cartoDB). Other options are MapBox (has free plan) and LeafletJS (free) and StoryMap (part of Knight Lab tools - Juxtapose, SoundCite, StoryMap and Timeline).
  • Data Visualization for all - Tell your data story with free and easy-to-learn tools
  • ** Infogram: create charts and infographics online. Similar tools include easel.ly (supports Arabic), Pictochart (see examples), visual.ly, Visme, Venngage, ** Venngage and ** Ceros (interactive).
  • Data Driven Guides using this online tool (exports as SVG)
  • Cool Infographics - many tools in different categories
  • Knight Lab projects (storytelling)
  • Cytoscape: networkd analysis and visualization
  • Graph Commons: network mapping
  • Timeline.js: easily made timelines
  • Silk: online interactive visualization
  • Piktochart: easy to use infographics maker
  • Tableau Public: free version (limited) of commercial application - analysis and visualization (note: Tableau acquired InfoActive in Aug. 2015
  • Plotly: charts, visualization and dashboards
  • ** Canva: simple, customizable, general purpose design tool (free, online, does infographics, OK with Arabic). For tutorials, try Canva tutorials or this video or this or this (simple infographic). There is also a data infographic tutorial
  • Keshif: from Univ. of Maryland
  • Data GIF Maker from Google. For general purpose GIFs, try sites like Giphy or Ezgif
  • Data Visualization principles: The Curious Journalist’s Guide to Data and Data visualization: basic principles (part of Introduction to Data Visualization course). See also: Chart Chooser and Data Visualization catalogue.
  • Programming languages like Python and R (require coding) and other commercial software and services
  • See also (examples) at Visualizing Impact and their projects: Visualizing Palestine and Visualizing Egypt
  • Raw Graphs: from data to visualization, online.
  • Tweet sentiment visualization (use keyword or hashtags, does not work in Arabic), see also TrendsMap (visuals and analytics)
  • Microsoft text analysis tools service based, recognizes Arabic (not supported yet, Translate?). See also IBM Tools for language analysis (sentiment and emotion and more, Arabic not supported yet) and Google Cloud (sentiment and more, Arabic not supported yet)
  • Repustate text analytics (different languages including Arabic - limited support), see also Ist Networks, Aylien, Bitext, DatumBox, MeaningCloud (has Excel plugin)
  • List of social media analysis APIs, another list can be found in this Qoura question and more at Sentiment Analysis Tools **. Note: some services like Lexalytics have Excel plug-in (claims Arabic support - very limited).
  • Tweet Reach - Twitter analytics, enter word or hashtag or user and analyze
  • Linguistic Inquiry and Word Count: analyzes text linguistically and psychologically. This is what powers Analyze Words for Twitter (no obvious Arabic support).
  • Website analytics: examples include Similar Web, Alexa, GTmetrix (speed and performance), Visitor Detective, ... etc. Notice that not all sites may be covered, best is to use native site stats (logs) or a service like Google analytics.
  • Social media monitoring and analytics: Brand24 (brand reputation), TalkWalker
